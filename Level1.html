Довършете едноредовата инструкция cout между main() и return 0;.
Тя трябва да изведе точно Hello, C++ World! и да завършва с точка и запетая.

#include <iostream> и using namespace std; вече са добавени; можете да редактирате само този един ред.

#include <iostream>
using namespace std;
int main() {
    return 0;
}
